## 🚀 Setup

1. Install [Python](https://www.python.org/) >=3.7
2. Run `pip install atom-dl` as administrator. </br>
    <sup>(To upgrade from an older Version use `pip install -U atom-dl` instead)</sup>
4. Read the Usage section



### Usage

See `atom-dl -h`
